MICRO OVEN PROJECT

This project simulates a Microwave Oven using PIC16F887 microcontroller.

 Features
- Power mode
- Grill mode
- Convection mode
- Timer display on CLCD
- Matrix keypad input

 Software Used
- MPLAB X IDE
- XC8 Compiler
- PICSimLab

 Hardware Used
- PIC16F887
- CLCD (16x4)
- Matrix Keypad

 How to Run
1. Open project in MPLAB X IDE
2. Build the project
3. Load the HEX file into PICSimLab
4. Run simulation
